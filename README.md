# Purelyst WordPress Theme

A minimalist WordPress theme dedicated to clarity, design, and intentional living.

## Description

Purelyst is a modern, clean WordPress theme designed for bloggers, writers, and content creators who value simplicity and readability. Built with performance in mind, it achieves excellent Core Web Vitals scores while maintaining a beautiful, contemporary aesthetic.

## Features

- **Clean, Minimalist Design** - Focus on content with elegant typography and generous whitespace
- **Fully Responsive** - Looks great on all devices from desktop to mobile
- **Core Web Vitals Optimized** - Excellent LCP, CLS, and INP scores
- **Customizer Options** - Easy customization through WordPress Customizer
- **Dark Mode Support** - Automatic dark mode based on system preferences
- **SEO Friendly** - Clean, semantic HTML structure
- **Accessibility Ready** - WCAG compliant with proper ARIA labels and skip links
- **Translation Ready** - Full internationalization support

## Installation

1. Download the theme zip file
2. Go to WordPress Admin → Appearance → Themes
3. Click "Add New" → "Upload Theme"
4. Choose the downloaded zip file and click "Install Now"
5. Activate the theme

## Customization

### Theme Options

Navigate to **Appearance → Customize** to access:

- **Hero Section** - Configure featured post and hero content
- **Sidebar Settings** - Set up author widget and bio
- **Newsletter Widget** - Configure email signup form
- **Footer Settings** - Customize footer content
- **Social Links** - Add your social media profiles
- **Colors** - Customize primary and accent colors

### Menus

The theme supports three menu locations:

1. **Primary Menu** - Main navigation in header
2. **Footer Explore** - First footer menu column
3. **Footer Company** - Second footer menu column

### Widget Areas

1. **Sidebar** - Additional widgets below default sidebar content
2. **Footer Widget Area** - Custom widgets in footer
